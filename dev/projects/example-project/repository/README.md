# Example Project

This is a minimal example project for testing MANFRED.

## Files

- `hello.rb` - A simple Ruby script that can be modified by Claude Code
- `docker-compose.yml` - Docker compose configuration
- `Dockerfile` - Container image with Claude Code installed

## Running

This project is meant to be run via MANFRED:

```bash
bin/manfred job example-project prompts/test-prompt.txt
```
